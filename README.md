# Convene Floor Plan Tool

Interactive SVG floor plan viewer for AV operations at Convene Chicago.

## Quick Start

Open the app locally:

```bash
open index.html
```

## Structure

Single-file HTML app (~2300 lines). Vanilla JS, no build system, no frameworks.

- `index.html` — entire application
- `data/g2-conference.js` — event and room data
- `FLOORPLAN-REDESIGN-PLAN.md` — multi-agent improvement roadmap
- `DESIGN-RESEARCH-BRIEF.md` — UI/UX research findings

Map-first interface with detail sidebar. localStorage checklist persistence.
